# AutoTrading

Using MA (moving average) to predict trends.
Time periods used in MA are 5 days, 15days and 120 days.

* buy signals
  * price cross a MA
  * Golden cross

* sell signals
  * a MA cross price
  * Death cross
  
## Environment
* Ununtu 16.04
* Python 3.6.4
## Setup
  `pip install -r requirements.txt`
## Run
  `python trader.py`
